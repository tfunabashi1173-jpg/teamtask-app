## backend-copy

`backend-copy/` は、既存の PWA バックエンドから必要な API・認証・通知・Supabase SQL を
`teamtask-app` リポジトリ内へコピー保存した参照用ディレクトリです。

目的:

- ネイティブアプリ側の実装で必要になる API 仕様とサーバー処理を、このリポジトリ単体で確認できるようにする
- 将来的にバックエンドを別管理または再編成する際の移行元スナップショットを保持する
- まだ運用中の PWA 側ファイルを消さずに、同内容を複製して持つ

注意:

- ここにあるファイルは「移設」ではなく「コピー」です
- 現在の Expo アプリ本体のビルド対象には含めません
- PWA 側が引き続き本番運用に使うため、元ファイルは削除しません
